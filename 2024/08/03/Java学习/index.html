1. Java常用方法与类

1. String 类

  • charAt(int index): 返回指定索引处的字符。
  • toCharArray(String str): 获取对应的字符数组。
  • substring(int beginIndex, int endIndex): 返回一个新字符串，它是此字符串的一个子字符串。
  • length(): 返回字符串的长度。
  • trim(): 去掉首尾空格。
  • split(String regex): 根据给定正则表达式拆分此字符串（返回字符数组）。
  • toLowerCase(): 全部变成小写。
  • toUpperCase(): 全部变成大写。
  • indexOf(String str): 返回指定子字符串第一次出现的索引。
  • contains(String str): 是否包含子字符串。
  • valueOf(object a): 将指定数字变为字符串。
  • replaceAll(String old,String new): 替换所有的子字符串。
  • replaceFirst(String old,String new): 只替换第一个子字符串。

2. Character类（字符类）

  • isLetter(char c): 判断是否为字母
  • isDigit(char c): 判断是否为数字
  • isWhitespace(char c): 是否是空白
  • isUpperCase(char c): 是否是大写
  • isLowerCase(char c): 是否是小写
  • toUpperCase(char c): 转换为大写
  • toLowerCase(char c): 转换为小写

3. StringBuilder 类

  • append(...): 将任意类型的数据追加到字符串生成器的末尾。
  • toString(): 将此序列转换为一个字符串。
  • insert(int offset, ...): 在指定位置插入数据。
  • delete(int start,int end): 删除指定区段数据。
  • resverse(): 反转字符串。

4. Math 类

  • abs(int a): 返回绝对值。
  • max(int a, int b): 返回两个参数中的较大值。
  • min(int a, int b): 返回两个参数中的较小值。
  • pow(double a, double b): 返回第一个参数的第二个参数次幂。
  • sqrt(double a): 返回正确舍入的正平方根。
  • random(): 获得一个0-1之间的随机浮点数（取不到1）。

5. Arrays 类(数组类)

  • sort(int[] a): 对数组进行排序。
  • binarySearch(int[] a, int key): 使用二分搜索法来搜索数组（必须先用sort排序）。
  • equals(int[] a, int[] a2): 如果两个指定数组彼此相等，则返回 true。
  • fill(int[] a, int val): 将指定值填充到数组的每个位置。
  • copyOf(int[] original, int newLength): 复制数组。
  • copyOfRange(int[] original, int startIndex, int endIndex(取不到)): 复制数组。
  • toString(int[] a): 将数组转变为字符串格式。

6. Collections 类（容器工具类）

  • sort(List<T> list): 对列表进行排序**(适用于Arraylist)**。
  • binarySearch(List<? extends Comparable<? super T>> list, T key): 使用二分搜索法来搜索列表。
  • reverse(List<?> list): 反转列表中元素的顺序。
  • swap(List<?> list,int indexa,int indexb): 交换列表中两个位置的元素。
  • rotate(List<?> list,int len): 把List中的数据，向右滚动指定单位的长度。
  • shuffle(List<?> list): 使用默认随机源随机排列指定列表中的元素。

7. List 接口（ArrayList、LinkedList）

  • add(E e): 将元素添加到列表的末尾。
  • get(int index): 返回列表中指定位置的元素。
  • set(int index, E element): 用指定元素替换列表中指定位置的元素。
  • remove(int index): 移除列表中指定位置的元素。
  • size(): 返回列表中的元素数。
  • index of(object a): 查找元素的索引（未找到返回-1）。

8. Set 接口（HashSet、TreeSet）

  • add(E e): 将指定的元素添加到集合中（如果尚未存在）。
  • contains(Object o): 如果此集合包含指定的元素，则返回 true。
  • remove(Object o): 如果存在，则从集合中移除指定的元素。
  • size(): 返回集合中的元素数。

9. Map 接口（HashMap、TreeMap）

  • put(K key, V value): 将指定的值与此映射中的指定键相关联。
  • get(Object key): 返回指定键所映射的值。
  • remove(Object key): 如果存在，则移除此映射中指定键的映射关系。
  • containsKey(Object key): 如果此映射包含指定键的映射关系，则返回 true。
  • containsValue(Object value): 如果此映射包含指定值的映射关系，则返回 true。
  • keySet(): 返回此映射中包含的键的 Set 视图。

10. System 类

  • arraycopy(Object src, int srcPos, Object dest, int destPos, int length): 从指定源数组中复制一个数组，复制从指定的位置开始到目标数组的指定位置结束。
  • currentTimeMillis(): 和new Date().getTime()相同，当前日期的毫秒数。

11. Exception 处理

  • try-catch: 用于捕获和处理异常。
  • throw new Exception(...): 用于手动抛出异常。
  • finally: 始终执行的代码块，用于释放资源。

4. 位操作

左移：根据一个整数的二进制表达，将其每一位都向左移动，最右边一位补0
右移：根据一个整数的二进制表达，将其每一位都向右移动

带符号右移 >>
对于正数， 带符号右移 >> 会把所有的位右移，并在最前面补0
对于负数， 带符号右移 >> 会把所有的位右移，并在最前面补1
无符号右移>>>
如果是一个负数，那么对应的二进制的第一位是1
无符号右移>>>会把第一位的1也向右移动，导致移动后，第一位变成0
这样就会使得负数在无符号右移后，得到一个正数

简单的说：
带符号右移 >> 移动后正的还是正的，负的还是负的,符号不变
无符号右移>>>移动后，变正的了

5. 多态

多态: 都是同一个类型，调用同一个方法，却能呈现不同的状态

要实现类的多态，需要如下条件：

  1. 父类（接口）引用指向子类对象

  2. 调用的方法有[重写]

静态方法与类直接关联，而不是与实例对象关联。多态通常是在实例方法上发挥作用，因为实例方法的调用是基于对象的运行时类型来决定的。但是静态方法是直接通过类名调用的，与实例对象无关，实际上，类方法并不具有多态性，也不会被子类的重写所影响

8. 异常

运行时异常可以被捕捉，error不能被捕捉
运行时异常与非运行时异常的区别：运行时异常是非受检查异常，它可以在代码中不必显示的处理或抛出；非运行时异常都是受检查异常，必须在代码中显式地处理或者抛出。如果不处理或抛出，则编译器会报错。

自定义异常：

  1. 创建一个自定义Exception实例
  1. 通过throw 抛出该异常
  1. 当前方法通过 throws 抛出该异常

try():

把流定义在try()里,try,catch或者finally结束的时候，会自动关闭
这种编写代码的方式叫做 try-with-resources， 这是从JDK7开始支持的技术

所有的流，都实现了一个接口叫做 AutoCloseable，任何类实现了这个接口，都可以在try()中进行实例化。 并且在try, catch, finally结束的时候自动关闭，回收相关资源。
